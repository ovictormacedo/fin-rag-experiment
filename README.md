# Experiment using FAISS to demo RAG

### Requirements
* python3.11+
* pip
* gemma3:4b running locally using Ollama, though you may adapt the code to call some external API

### Create a virtual environment
```sh
python3.11 -m venv venv
```

### Enable the virtual environment
https://docs.python.org/3/library/venv.html#how-venvs-work

### Install the libs
```sh
pip install -r requirements.py
```

### Run the API
```sh
python api.py
```

### Open the interface
Open frontend/chat.html on any browser
